Olá esse projeto faz parte do FreeCodeCamp, é o quarto projeto deles do curso de "Resposive Web Design Certification", onde era pra ser feito um site landing page, com formulário, menu de navegação e algumas estilizações exigidas, incluindo responsividade. Fiz ele como se fosse um site de uma organização de adoção e resgate de cachorros e deixei ele responsivo em diversos tipos de telas, incluindo um meno retrátil usando javascript e css.
